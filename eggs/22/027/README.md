**Project description and instructions**  
This egg contains the [GROMACS](https://www.gromacs.org) topology and input files as well as the PLUMED/python analysis scripts needed to reproduce the Molecular Dynamics simulations of the WT and mutant BANAL-236 spike-RBD/human-ACE2 complexes. See [README.md](https://github.com/maxbonomi/BANAL-236/#readme) for more information and detailed instructions.
